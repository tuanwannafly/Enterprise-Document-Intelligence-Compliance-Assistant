# Utility scripts

Convenience wrappers for common developer workflows.

* `seed_demo.py` — seed the local stack with synthetic demo documents and run the full ingestion pipeline.
* `run_tests.py` — run the test suite with the correct `PYTHONPATH`.
* `bootstrap-db.sh` — initialise a fresh Postgres instance with the `edi_owner` + `edi_app` roles.
